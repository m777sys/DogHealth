# Pet Wellness Hub

Dog health products and supplements affiliate site.

## GitHub Pages Setup Required

To make this site live:

1. Go to Settings → Pages
2. Source: Deploy from a branch
3. Branch: main
4. Folder: /docs
5. Save

Site will be live at: https://m777sys.github.io/DogHealth/

- `docs/index.html` - Main page
